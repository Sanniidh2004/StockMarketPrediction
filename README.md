## AD_Lab_2330044_Experiment-2(A)
Building a Stock Market predictor for different market organisations using html (frontend) and python flask (backend)  – Application Development Lab
### Project Description
This project is a stock market prediction webpage created using HTML and Python as part of the AppDev Lab assignment. It can import data from Yahoo Finance and turn it into graphical visuals and also predict future market value
### Technologies Used
- HTML5
- Flask
- Visual Studio Code
- GitHub
### Files Included
- index.html - Webpage structure and layout
- train_models.py – Training LSTM and Linear Regression models for prediction
- app.py – Backend code
